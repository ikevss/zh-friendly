# 中英术语对照表

> 本表用于统一汉化时的技术术语翻译风格。翻译 description 和正文时，按此表处理。

## 规则

1. 标注"不翻译"的术语，在任何上下文中都保留英文原文
2. 标注"中英双写"的术语，格式为"中文（English）"
3. 标注"视上下文"的术语，根据语境选择中文或英文
4. 未收录的术语，优先保留英文原文，除非中文译法已是行业共识
