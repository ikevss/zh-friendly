常见问题

汉化后技能还能正常触发吗？

能。翻译时保留了英文触发关键词，而且加了中文口语表达，触发率反而更高。已在新机器上实测跑通。

会改坏我的 Agent 人设吗？

不会。Agent 文件的正文（System Prompt）一个字都不动。翻译完逐字符校验——多一个空格都会从备份恢复。

翻译完以后更新工具会被覆盖吗？

marketplace 来源的可能会。用 --overlay 模式可以避免：不改原 description，新加一个 description_zh 字段。

支持哪些 AI 工具？

Claude Code、TRAE Work、Codex、Cursor、OpenCode、Windsurf、WorkBuddy、Antigravity、OpenClaw。更多工具持续适配中。
