不是机翻。是把你可能会说的中文表达加进去。比如 Deployment 不只翻译成"部署"，还会加上"帮我发布""上线""推代码"——你说哪个它都能懂。

你可能要问

常见问题

翻完还能用吗？

能。英文关键词原样保留，额外加了你可能会说的中文表达。原来能用的继续能用，你说中文也能触发。

不会把我的 Agent 改坏吧？

不会。你的 Agent 正文一个字不动。只改给人看的那一行 description。改完逐个字符校对，多一个空格都会恢复。

以后更新了会不会又变回英文？

有可能。但你加 --overlay 参数就不怕——不碰原文，新加一个中文字段。更新了也丢不了。

我其他的 AI 工具也管吗？

管。你装了哪个就扫哪个：TRAE Work、Codex、Cursor、Windsurf、WorkBuddy、OpenClaw。Antigravity 只读不写。

一个小编的故事

有一天我数了一下斜杠菜单——352 个技能，名字全是英文。每次都要点进去试一下才知道干嘛的。试对了就好，试错了白费一次。更离谱的是，AI 自己也不知道有多少能力——它连一份清单都没有。于是我写了这个工具。扫一遍，翻成中文，一眼看懂。踩过的坑都写进了 references/，五方工程师对抗性评审过。
