核心的就两件

事情是这样的。我装了 352 个技能，每次打开斜杠菜单跟猜谜一样——这个名字大概管画图？那个好像是部署？得点进去试。试对了就好了。试错了就浪费一个来回。于是我写了这个，扫一遍，翻译成中文，一眼看懂。

扩写式翻译

不是机器直译。每个描述里加载 3-5 个你可能说的中文表达。英文触发词保留，中文口语扩展。

常见问题

翻完还能正常用吗？

能。英文关键词原样保留，额外加了中文口语表达。实测过了，原来英文能触发的依然能，你用中文说也能。

不会把 Agent 改坏吧？

Agent 正文一个字不动。只改给人看的那行 description。改完还会逐字符校验——多了一个空格都会自动从备份恢复。

更新了会不会又变回英文？

marketplace 来源的有可能。加 --overlay 就行——不碰原 description，新加一个 description_zh 字段存中文。

其他 AI 工具也管吗？

管。TRAE Work、Codex、Cursor、Windsurf、WorkBuddy、OpenClaw——电脑上有哪个扫哪个。Antigravity 例外，那个只读不写。
