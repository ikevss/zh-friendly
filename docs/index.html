常见问题

汉化后技能还能正常触发吗？

能。翻译时保留英文关键词，额外加了中文口语表达。原来能用的继续能用，你说中文也能触发。

会不会改坏我的 Agent？

不会。Agent 文件的 System Prompt 正文一个字不动，只改给人看的 description。改完逐字符校验。

更新了工具会不会覆盖汉化？

marketplace 来源的可能会。加 --overlay 参数可以不改原 description，新增 description_zh 字段。

只想汉化某个工具？

加 --scope 参数：/ikevss-zh-friendly zh --scope trae
